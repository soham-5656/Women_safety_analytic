Required Python Libraries: OpenCV pip install opencv-python argparse pip install argparse Usage : Download the repository. Open Command Prompt or Terminal and navigate to the folder containing all files. To detect gender and age in an image, use the command: python detect.py --image Note: The image should be in the same folder as the files. To detect gender and age through the webcam, use the command: python detect.py Press Ctrl + C to stop program execution.
